\projectentry{Indoor Mapping and Navigation System using Raspberry Pi 5}{2026}
{Research Project | IIT Tirupati Navavishkar i-Hub Foundation (IITTNiF) \& CUSAT}
{
    \item Developed a low-cost indoor localization and mapping system using Raspberry Pi 5, monocular camera, IMU, and Time-of-Flight (ToF) LiDAR sensors.
    \item Implemented Visual-Inertial SLAM with multi-sensor fusion to generate real-time 2D occupancy maps for GPS-denied indoor environments.
    \item Designed a QR-code-based navigation framework enabling users to access interactive indoor maps and shortest-path navigation through a web interface.
    \item Improved localization accuracy by integrating ToF-based range validation, significantly reducing trajectory drift during indoor mapping experiments.
}

\projectentry{Audio-Band FSK Text Modem using GNU Radio}{2026}
{SDR Project - DOE, CUSAT}
{
    \item Designed and implemented an audio-band Frequency Shift Keying (FSK) modem using GNU Radio to transmit text data through standard computer microphones and speakers.
    \item Developed complete transmitter and receiver flowgraphs incorporating modulation, demodulation, quadrature demodulation, binary slicing, and file reconstruction.
    \item Implemented reliable end-to-end text communication by converting digital data into audio tones and recovering the transmitted message in real time.
    \item Evaluated modem performance under varying channel conditions and optimized parameters to improve decoding accuracy and communication reliability.
}

\projectentry{SDR-Based Anti-Jamming Audio Communication System}{2026}
{SDR Project - DOE, CUSAT}
{
    \item Developed a full-duplex Software Defined Radio communication system using the Ettus B210 to investigate interference mitigation in wireless audio transmission.
    \item Simulated intentional wideband noise jamming while transmitting Narrowband FM (NBFM) audio over independent SDR channels.
    \item Designed adaptive software filtering techniques to suppress interference and recover intelligible audio under high jammer power conditions.
    \item Analyzed receiver performance by varying jammer intensity, filter bandwidth, and signal-to-noise ratio (SNR) to evaluate anti-jamming effectiveness.
}
