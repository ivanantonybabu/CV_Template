# 📄 Professional LaTeX CV Template

A clean, modern, and ATS-friendly CV template built using **LaTeX** and designed on **Overleaf**.

## 🚀 Getting Started

- Clone or fork this repository.
👉 **[Open this project in Overleaf](https://www.overleaf.com/project)**- Edit the **`main.tex`** file with your own details.
- Compile and download your professional CV.

## ✨ Customization

No need to modify the template structure—simply update the **`.tex`** file according to your requirements.

If this template helped you, consider leaving a ⭐ on the repository!

Happy TeXing! 🚀
